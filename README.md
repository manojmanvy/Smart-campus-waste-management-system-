# Smart-campus-waste-management-system-
SCWMS using IoT Designed and programmed an IoT-based system using Arduino IDE and ESP32 to monitor garbage levels in real-time. Used ultrasonic sensor, DHT22, LCD, buzzer, and LED for sensing and alerting. Integrated Blynk mobile app for remote notifications. Focused on clean coding, hardware-software integration, and real-time data handling.
